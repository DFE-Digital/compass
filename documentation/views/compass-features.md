# COMPASS Features

COMPASS (Compliance, Outcomes, Performance, Assurance, Standards and Strategy) is a comprehensive platform designed to streamline government service reporting, milestone tracking, and administrative management.

## Overview

COMPASS provides a unified interface for managing government service compliance, performance reporting, and administrative tasks. The platform is built with modern web technologies and follows government digital service standards.

## Platform modules

COMPASS consists of three main modules that work together to provide comprehensive service management:

### Admin module
- **Create and manage reporting metrics**: Configure and manage performance indicators for services
- **Manage admin users**: Control who can create and manage metrics within the system
- **Create and manage objectives**: Define strategic objectives that are assigned to milestones

[View detailed Admin features →](compass-admin.md)

### Reporting module
- **Monthly performance reporting**: Submit and manage monthly performance returns for assigned services
- **Service overview dashboard**: View all services assigned to the user with reporting status
- **Performance metrics completion**: Complete predefined performance indicators with data validation
- **Report submission management**: Submit individual service reports or bulk submissions

[View detailed Reporting features →](compass-reporting.md)

### Milestones module
- **Milestone management**: Create and configure service delivery milestones with titles, descriptions, and target dates
- **Objective linking**: Associate milestones with strategic objectives created in Admin module
- **Progress tracking**: Monitor milestone progress with RAG status indicators (Red, Amber, Green)
- **Timeline management**: Set and track milestone deadlines and dependencies

[View detailed Milestones features →](compass-milestones.md)
