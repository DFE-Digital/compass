# Skills and Learning Module - Navigation Guide

## Overview

This guide explains where to find all Skills and Learning features in the navigation menus.

## 📍 Course Management

### Location: Admin Area
**Path:** `/Admin/TrainingCourse`

**Access:** Admin users (DesignOps)

**Navigation:**
1. Go to **Admin** section (top navigation or sidebar)
2. Look for **"Learning & Development"** section
3. Click **"Training courses"**

**Features Available:**
- View all courses (with search and active/inactive filter)
- Create new course
- Edit existing course
- Archive course (soft delete)
- Restore archived course
- View course details

**Direct URL:** `/Admin/TrainingCourse`

---

## 💰 Budget Management

### Location: Skills and Learning Section
**Path:** `/HOP/BudgetAndSpending` → Click "Manage budget" button

**Access:** Central Operations Admin only

**Navigation:**
1. Go to **Skills and Learning** section (main sidebar)
2. Click **"Budget and spending"** (visible to HOP and Central Ops Admin)
3. Click **"Manage budget"** button (top right, Central Ops Admin only)

**Alternative Navigation:**
- Direct link: `/HOP/ManageBudget` (Central Ops Admin only)

**Features Available:**
- View current financial year budget
- Set/update total L&D budget amount
- View actual spent, forecasted costs, and remaining budget
- Budget automatically calculated from training records

**Direct URL:** `/HOP/ManageBudget`

## 📊 Other Features Locations

### Individual User Features
**Navigation:** Skills and Learning → (submenu)
- **My skills and learning** - Dashboard with recommendations
- **Browse courses** - Browse available training
- **My requests** - View and manage training requests
- **Learning history** - Track completed training

### Learning & Skills Role Features
**Navigation:** Skills and Learning → (submenu)
- **Profession requests** - Approve/reject requests
- **View learning** - View all learning records
- **Manage skills and learning** - Admin functions
- **L&D Reporting** - Analytics and reports

### HOP/Central Ops Admin Features
**Navigation:** Skills and Learning → (submenu)
- **Profession requests** - View profession-wide requests
- **Budget and spending** - Budget dashboard
- **L&D Reporting** - Analytics and reports

### Reporting Features
**Navigation:** Skills and Learning → L&D Reporting
- **Main Dashboard** - Overview with KPIs and charts
- **Spend Analysis** - Detailed spending breakdown
- **Outcomes & Satisfaction** - Completion rates and ratings
- **Profession Analytics** - Profession-level statistics
