# COMPASS Milestones

COMPASS (Compliance, Outcomes, Performance, Assurance, Standards and Strategy) Milestones module provides comprehensive tracking and management of service delivery milestones and objectives.

## Milestones module

### Key features
- **Milestone management**: Create and configure service delivery milestones with titles, descriptions, and target dates
- **Objective linking**: Associate milestones with strategic objectives created in Admin module
- **Progress tracking**: Monitor milestone progress with RAG status indicators (Red, Amber, Green)
- **Timeline management**: Set and track milestone deadlines and dependencies
- **Status updates**: Add milestone updates with status changes and progress comments
- **Priority management**: Set milestone priorities (High, Medium, Low)
- **Overdue tracking**: Identify and manage overdue milestones
- **Milestone history**: Track milestone updates and status changes over time
