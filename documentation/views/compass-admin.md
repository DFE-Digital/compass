# COMPASS Admin

COMPASS (Compliance, Outcomes, Performance, Assurance, Standards and Strategy) Admin module provides administrative capabilities for system management and user administration.

## Admin module

### Key features
- **Create and manage reporting metrics**: Configure and manage performance indicators for services
- **Manage admin users**: Control who can create and manage metrics within the system
- **Create and manage objectives**: Define strategic objectives that are assigned to milestones

## Admin dashboard

### Key features
- **System overview**: View total users, active users, total metrics, active metrics, total milestones, overdue milestones, total products, and monthly submissions
- **Statistics tracking**: Monitor system usage and performance metrics
- **Quick access**: Direct access to admin functions

## User management

### Key features
- **User permissions**: Create and manage user permissions with granular access control
- **Permission settings**: Configure permissions for products, metrics, milestones, and reports
- **User roles**: Set permissions for adding, editing, deleting, and viewing different system components
- **Active user management**: Enable and disable user accounts

## Performance metrics management

### Key features
- **Create metrics**: Add new performance metrics with names, descriptions, and phase assignments
- **Edit metrics**: Update existing performance metrics
- **Delete metrics**: Remove performance metrics from the system
- **Phase assignment**: Assign metrics to specific service phases (Discovery, Alpha, Beta, Live, Retired)
- **Metric configuration**: Set legal/regulatory flags and mandate requirements

## Objectives management

### Key features
- **Create objectives**: Define new strategic objectives
- **Edit objectives**: Update existing objectives
- **Delete objectives**: Remove objectives from the system
- **Objective details**: View detailed information about objectives
- **Milestone linking**: Objectives can be assigned to milestones
