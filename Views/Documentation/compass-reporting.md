# COMPASS Reporting

COMPASS (Compliance, Outcomes, Performance, Assurance, Standards and Strategy) Reporting module enables users to submit and manage monthly performance returns for their assigned services.

## Users

People who have the role on a product as **Reporting** or **Service owner** will be permitted to complete monthly reports for a product.

This is recorded in the CMS against the products or services in the Product_Contacts relationship. See the [Product Contact model documentation](http://localhost:3000/#data-models#13-product-contact-model) for detailed information about the Product_Contacts relationship and data structure.

## Reporting module

### Key features
- **Monthly performance reporting**: Submit and manage monthly performance returns for assigned services
- **Service overview dashboard**: View all services assigned to the user with reporting status
- **Performance metrics completion**: Complete predefined performance indicators with data validation
- **Report submission management**: Submit individual service reports or bulk submissions
- **Reporting status tracking**: Track completion status, due dates, and submission status
- **RAG status monitoring**: Monitor Red, Amber, Green status indicators for service health

## Dashboard

### Key features
- **Service overview**: View all services assigned to the user
- **Reporting status**: Track completion status of monthly returns
- **Due date tracking**: Monitor upcoming and overdue reporting deadlines
- **Quick actions**: Direct access to reporting tasks

## Monthly performance reporting

### Key features
- **Service selection**: Choose from assigned services for reporting
- **Performance metrics**: Complete predefined performance indicators
- **Data validation**: Built-in validation for metric values and formats
- **Progress tracking**: Visual indicators for completion status
- **Submission management**: Submit individual service reports or bulk submissions
